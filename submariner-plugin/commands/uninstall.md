---
description: Uninstall Submariner components
---

Safely remove Submariner components from the cluster using subctl.

Execute `subctl uninstall` to:
1. Confirm the user wants to proceed with uninstallation
2. Remove Submariner components from the cluster
3. Clean up associated resources
4. Verify complete removal

Warn about the impact of uninstalling Submariner on cross-cluster connectivity.
